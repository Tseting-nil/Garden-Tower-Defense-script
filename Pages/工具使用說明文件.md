# 🏰 塔放置 & 升級追蹤工具 使用說明

## 📋 工具簡介

此工具專為「花園塔房 (Garden Tower Defense)」遊戲設計，能夠**自動追蹤並記錄**您在遊戲中的所有操作，包括：

- 🗼 放置塔的位置與時間
- ⬆️ 升級塔的時間點
- ⏭️ 跳過波次 (Skip Wave)
- 💰 出售塔
- ⚡ 遊戲速度調整 (1x/2x/3x)
- 🔄 AutoSkip 開關切換

記錄完成後，工具會**自動生成可執行的腳本**，讓您可以重複執行相同的操作序列。

---

## 🚀 快速開始

### 1️⃣ 執行工具

在遊戲中執行腳本，工具介面會自動顯示。

### 2️⃣ 進入遊戲

1. 在大廳選擇您要遊玩的地圖
2. 進入遊戲後，選擇難度（Easy/Normal/Hard/Insane/Impossible/Apocalypse）
3. 工具會自動偵測遊戲倒數，當倒數開始時自動啟動計時器

### 3️⃣ 正常遊玩

像平常一樣遊玩遊戲，**所有操作都會被自動記錄！**

### 4️⃣ 複製腳本

遊戲結束後，點擊「📋 複製腳本」按鈕，生成的自動化腳本會複製到剪貼板。

---

## 🎮 介面功能說明

### 主介面

| 元素 | 說明 |
|------|------|
| **標題欄** | 可拖曳移動整個視窗 |
| **日誌區域** | 顯示所有追蹤到的操作記錄 |
| **📋 複製腳本** | 生成並複製自動化腳本到剪貼板 |
| **⚙️ 參數** | 開啟參數設定面板 |
| **- 按鈕** | 收合/展開主介面 |

### 參數設定面板

| 設定項目 | 說明 |
|----------|------|
| **腳本生成倍率** | 可將 1 倍速遊玩的記錄轉換為 2x 或 3x 倍速腳本 |
| **時間秒數自動進位** | 開啟時會將時間進位處理 |
| **放置塔自動關閉介面** | 生成的腳本是否自動關閉升級介面 |
| **自動重新執行** | 遊戲結束後是否自動重新執行腳本 |
| **自訂註釋內容** | 生成腳本的註釋說明文字 |
| **自動捲軸** | 日誌區域是否自動捲動到最新記錄 |
| **重置** | 清除所有追蹤記錄重新開始 |
| **查看綁定** | 顯示目前已追蹤的塔列表 |

---

## ⌨️ 快捷鍵

| 按鍵 | 功能 |
|------|------|
| **F8** | 顯示/隱藏工具介面 |

---

## ⚠️ 注意事項
> **⚠️ 此腳本僅供學習與研究用途**
1. **等待遊戲開始** - 選擇難度後，請等待「✅ 遊戲開始！」訊息出現
2. **腳本生成倍率** - 轉換後的腳本可能需要微調
3. **多人模式** - 工具只會追蹤您自己放置的塔
4. **路徑型塔** - 支援追蹤路徑型塔（如 Patrol 單位）

---

## 🔧 疑難排解

| 問題 | 解決方法 |
|------|----------|
| 介面不見了 | 按下 **F8** 鍵 |
| 腳本沒有複製成功 | 查看控制台 (F9 或 聊天室輸入 "/console") |
| 升級顯示「未追蹤」| 可能是其他玩家的塔 |
